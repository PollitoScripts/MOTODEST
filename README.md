# MOTODEST Barbate - Sitio Web Oficial 

Este es el repositorio del sitio web oficial para **MOTODEST**, el concesionario y taller de motocicletas de referencia situado en Barbate (Cádiz).
---

El objetivo de este proyecto es ofrecer una plataforma moderna, intuitiva y rápida para que los clientes puedan conocer el catálogo de motos (nuevas y de ocasión), consultar los servicios de taller y contactar directamente con el negocio.

---

##  Características Principales

*   **Diseño Responsive:** Optimizado al 100% para móviles, tablets y ordenadores.
*   **Catálogo Digital:** Sección dedicada a la exposición de motocicletas disponibles, especificaciones técnicas y precios.
*   **Reserva de Citas:** Formulario o acceso directo para solicitar mantenimiento o revisiones en el taller.
*   **Localización y Contacto:** Integración con Google Maps para ubicar fácilmente el local en Barbate y botones de acceso rápido a WhatsApp/Teléfono.
*   **SEO Local:** Configuración optimizada para que los moteros de la zona de la Janda encuentren el negocio fácilmente en Google.
